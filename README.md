# Edge Google Fix (MV3)
Redirects Bing search results to Google using declarativeNetRequest.

## Install
1. Download the ZIP from the Releases page.
2. Go to edge://extensions/
3. Enable Developer Mode.
4. Click “Load unpacked” and select the `extension/` folder.

## What it does
- Intercepts Bing search pages
- Redirects them to Google (keeps your `q=` query intact)

## Popup UI
- Shows redirect rule status
- Shows last redirect timestamp
- Run self-check button

## Privacy
This extension:
- Does not collect or send any data
- Only reads navigation events for Bing
